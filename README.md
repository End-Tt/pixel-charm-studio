# Pixel Charm Studio

A multilingual static website for a custom pixel bead / fuse bead DIY kit concept.

## Pages

- `index.html` - homepage
- `custom.html` - custom photo-to-pattern process
- `kits.html` - product kit overview
- `pricing.html` - launch pricing
- `order.html` - mock order builder
- `login.html` / `register.html` / `account.html` - static demo account flow

## Notes

The login and registration flow is a static MVP. It stores demo users in browser `localStorage`, which is suitable for sharing and validating the user flow, but not for real customer accounts or payments.
